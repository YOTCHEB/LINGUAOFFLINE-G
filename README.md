# LINGUAOFFLINE-AI
LinguaOffline AI is a multilingual offline translation system designed to translate text and audio between English, Chichewa, kirundi, swahili, lingala, and French. It uses lightweight AI models that run locally on computers, enabling fast and reliable translations without internet. The project includes a user-friendly interface, offline text and audio translation engines.

See `TEAM_INSTRUCTIONS.md` for roles, folders and deliverables.
